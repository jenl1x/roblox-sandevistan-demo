# Sandevistan Ability for Roblox
A script that simulates the Sandevistan ability from Cyberpunk: Edgerunners anime - slows down time for all NPCs while keeping the player fast.

* Server logic: handles event requests and slows down NPCs
* Client effects: adds time-slow visuals and sounds
* Animations: smoothly adjusts animation speed
* Performance: optimized loops, no lag spikes

## Technologies used
Lua, Roblox Studio, RemoteEvents, ModuleScripts, Animator API
